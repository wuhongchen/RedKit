# Product-Spec CHANGELOG

## v1.4.0 (2026-02-15)

### 新增 (Added)
- **多场景列表支持**：新增对小红书首页、关键词搜索页、用户个人主页的适配，支持一键抓取笔记列表元数据。
- **自动化逐个提取**：实现“穿透式”采集逻辑，支持从列表自动进入详情页提取文案、评论及素材，并自动返回继续下一项。
- **数据汇总导出**：CSV 导出功能升级，支持详情数据与批量采集结果的合并汇总。
- **SPA 路由监听优化**：针对小红书单页应用架构，优化了 UI 面板在不同页面间的切换逻辑。

## v1.0.0 (2026-02-12)

### 新增 (Added)
- 项目初始化：明确项目名称为 XH-Scraper。
- 核心功能定义：包含浮动按钮抓取、作者 UID 提取、CSV 批量导出。
- 交互逻辑确定：采用侧边浮动大按钮全屏抓取 + 暂存导出模式。
- AI 增强建议：新增“爆款逻辑萃取”功能，支持分析笔记的钩子和情绪价值。
- 异常处理机制：确定列表页内容缺失时穿透到详情页进行补偿抓取。
